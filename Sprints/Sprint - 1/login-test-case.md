# Test case - *Login Screen*

## **Description**
- ### Testar a ação do usuario colocar email e senha para fazer login no site,  onde caso ele insira corretamente todos os dados, seja redirecionado para a proxima pagina.

## **Pre-conditions**
- ### 1 - Usuario deve ter email existente
- ### 2 - Senha deve atender os requisitos 
- ### 3 - Aceitar Termos e Condições
- ### 4 - Passar CapTcha

## **Steps to Reproduce**
- ### 1 - Usuario coloca email
- ### 2 - Usuario coloca senha
- ### 3 - Termos e Condições aceitos
- ### 4 - CapTcha feito com sucesaso

## **Expected Result**
- ### 1 - Usuario consegue fazer login
- ### 2 - Usuario entra ao site 
- ### 3 - Conta do usuario salvo
