## Day 1 - VSCode 

### Nesse dia nós instalamos a ferramenta chamada VSCode, ao qual é uma ferramente para programação, ou seja, formatação de textos como python, java, c++ e outros. Também aprendemos a baixar extenções que iram nós ajudar, são elas:Material Icon Theme, Markdown All in One , Intellicode. Após informamos tudo ao Daily-Checkin.


## Day 2 - Git e GitHub

### Aqui nós aprendemos a baixar o Git e usar-mos o GitHub(juntamente o Git), Nós criamos nosso primeiro reepositório, e vinculamos nossa primeira pasta cujo havia nosso nome(Exemplo: Enalab-Hiro). Além de criar-mos a nossa conta no GitHub, antes de criar o repositorio e informarmos novamente ao Daily-Checkin. Git é basicamente onde fazesmos nossos commit para "subir" nossos projetos para nosso repositório. GitHub é um "banco de dados" onde programadores colocam seus projetos na nuvem para outros possam utiliza-los, ou apenas para deixa-lo salvo sem ninghuem saber.

## Day 3 - MarkDown(Arquivo README.md)

### Neste dia, aprende-mos como funciona os arquivos README juntamente do que ser os arquivos markdown. Cria-mos nosso propio README e colocamos nossas informações pessoais assim entendendo como funciona os arquivos MarkDown e os README. README é o arquivo de informações sobre o projeto ou sobre vc mesmo, já MarkDown é o tipo de arquivo aparentemente mais facil de se utilizar.
